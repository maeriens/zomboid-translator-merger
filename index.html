Project Zomboid Missing Translations Helper

A helper that grabs an uploaded file, searches the English counterpart and downloads a merge between both, keeping the uploaded files translated lines but also adding the untranslated ones. The missing translations will also be listed in the text field below.

For example, uploading the UI_AR.txt file will search for the UI_EN.txt file in the translations github repository, prompt the user to download the merge file (named MERGED_UI_AR.txt) and then will show untranslated fields.

NOTE: files inside each city folder, the Survival Guide file and the language.txt file are not supported for direct translation.
